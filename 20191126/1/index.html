Какие преимущества дает bootstrap?

Bootstrap позволяет сэкономить время при написании кода, так как предоставляет возможность пользоваться шаблонами для создание различных элементов сайта. Эти шаблоны являются адаптивными.

В чем недостаки использования bootstrap?

Основным недостатком является то, что сайты, написанные на bootstrap'e друг на друга достаточно сильно похожи

Опишите особенности bootstrap

CSS Buttons

Существует несколько встроенных шаблонов кнопок, они имеют разный цвет, используются для разных целей. Также легко можно поменять цвет заливки кнопки, контур, ее размер, анимацию при клике.

CSS Tables

При использовании класса .table Bootstrap форматирует таблицу, придавая ей достаточно привлекательный вид. С помощью системы класоов в таблицах можно редактировать размер, цвет, выделение и прочие свойства

CSS Images

Bootstrap делает изображения отзывчивыми с помощью класа .img-fluid и max-width:100%; height:auto; таким образом изображение масштабируется по родительскому элементу

CSS Forms

Bootstrap с помощью классов и типов, указанных для элементов ввода позволяет создавать отзывчивые поля для ввода текста, e-mail, номера телефона, выбора изображения и т.д

CSS Grids

CSS-Grids - система сеток для расположения контента на сайте. Ячейки сетки могут быть фиксированными, или гибкими, а элементы размещаются в них, указанием номеры этой ячейки
